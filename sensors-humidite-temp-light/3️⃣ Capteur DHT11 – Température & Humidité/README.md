# DHT11 Sensor – ESP32 Project

## Description
Ce code lit la **température et l’humidité** avec le capteur DHT11 et affiche les données sur le moniteur série.

## Matériel
- ESP32
- DHT11
- Câbles de connexion

## Connexion des composants
| DHT11 | ESP32 |
|-------|-------|
| VCC | 3.3V |
| DATA | GPIO 15 |
| GND | GND |

## Fonctionnement
1. Démarrer le capteur DHT11  
2. Lire la température et l’humidité  
3. Afficher les valeurs sur le moniteur série  
4. Lecture toutes les 2 secondes
